О курсе

На курсе Fullstack-разработки на JavaScript изучаем профессиональные инструменты
front-end и back-end разработки React и Node.js, СУБД MySQL и MongoDB. Требования к
слушателям на входе: знания html, css, основ программирования на javascript.

На курсе преподают эксперты из IT-компаний Lad. В конце обучения слушатели курса
защищают итоговые проекты. Лучшие получают приглашение поступить на
стажировку, в интернатуру или в штат IT-компании.

Мы изучаем:

  • React

  • React Redux

  • Node JS

  • Git

  • Mongo DB

  • MySQL

Кому подойдет этот курс

  • Начинающим JavaScript-разработчикам

  • Начинающим разработчикам на React

  • Начинающим Backend-разработчикам

Требования к курсу

  • Знания HTML/CSS

  • Базовые знания JavaScript

  • Знания MySQL (будет плюсом)

  • Базовые навыки работы с Git (будет плюсом)

Карьера

Выпускники курсов HiBrain получают уникальную возможность применить новые знания:
компании-партнёры проекта ждут наших слушателей на стажировки и предлагают трудоустройство.

Программа курса

1 / Front-end разработка

Пишем браузерную часть web-приложения на React. Разбираем инициализацию
react-приложения, структуру проекта и работу с npm, виртуальный DOM,
компоненты, props и states, особенности потока данных, классовые и
функциональные компоненты, менеджер состояний Redux, роутинг и многое другое

2 / Back-end разработка

Пишем браузерную часть web-приложения на React. Разбираем инициализацию
react-приложения, структуру проекта и работу с npm, виртуальный DOM,
компоненты, props и states, особенности потока данных, классовые и
функциональные компоненты, менеджер состояний Redux, роутинг и многое другое

3 / Базы данных и СУБД

Пишем браузерную часть web-приложения на React. Разбираем инициализацию
